# CS 330 – Computational Graphics and Visualization
## 3D Scene Project Reflection

## Project Overview

In CS 330, I designed and developed a fully navigable 3D scene using C++ and OpenGL. This project required building 3D objects, applying textures, configuring lighting, and implementing camera controls to create an interactive environment. The final submission includes the 3D Scene ZIP folder and the Design Decisions document, which together demonstrate both technical execution and intentional design planning.

## Designing Software

### My Design Approach

When designing software, I begin by identifying the core objective and breaking it into manageable components. For this project, I focused first on scene structure, object placement, and user navigation before refining textures, materials, and lighting.

This project strengthened my ability to think spatially and architect systems visually. I learned how to plan object hierarchies, define transformations, and structure reusable functions to maintain clean and organized code.

### Design Process

I followed an incremental design process. Each milestone introduced new requirements such as camera movement, textures, or lighting. I implemented features step by step, tested functionality, and refined design decisions based on results. This structured progression ensured stability while gradually increasing scene complexity.

The same structured approach can be applied to future software projects by separating design, implementation, and refinement into clear phases.

## Developing Programs

### Development Strategy

When developing programs, I prioritize functionality first, then optimization and refinement. In this 3D scene, I created base geometry before adding textures and lighting effects. I also developed reusable rendering functions to reduce redundancy and improve maintainability.

Iteration played a major role in development. I frequently tested object scaling, UV mapping, lighting intensity, and camera positioning. Each adjustment improved realism and usability.

Throughout the milestones, my approach evolved from focusing on simply making objects render correctly to intentionally designing cohesive visuals with proper lighting balance, material definitions, and user navigation controls.

## Growth and Application

### Educational Pathway

Computational graphics expanded my understanding of linear algebra, transformations, lighting models, and spatial reasoning. These concepts strengthened my problem-solving skills and deepened my understanding of how mathematics directly supports software functionality.

### Professional Pathway

Computer science supports my long-term goals by giving me the technical foundation to design, build, and refine complex systems. Computational graphics added a visual and mathematical dimension to my skill set, strengthening both my analytical thinking and development discipline.
