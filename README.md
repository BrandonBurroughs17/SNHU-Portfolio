# CS 370 Portfolio Reflection

## Project Overview

For this project, I created an intelligent pirate agent that uses deep reinforcement learning to solve a maze and find a treasure. I was provided with the starter code for the maze environment, game mechanics, and supporting Python files. My responsibility was to complete the Q-training algorithm inside the Jupyter Notebook. I implemented the training loop, epsilon-greedy action selection, experience replay, neural network training, target network updates, and the logic to gradually reduce exploration as the model learned. After training, the pirate agent was able to consistently find the treasure, demonstrating that the reinforcement learning model had successfully learned an effective path through the maze.

## What Do Computer Scientists Do and Why Does It Matter?

Computer scientists solve real-world problems by designing algorithms, building software, and creating systems that improve efficiency and automate tasks. Their work affects nearly every industry, including healthcare, transportation, finance, manufacturing, entertainment, and defense. Artificial intelligence is becoming an increasingly important part of computer science because it allows systems to learn from data instead of relying only on fixed programming. This project demonstrated how machine learning can solve complex decision-making problems that would be difficult to program using traditional rule-based approaches.

## How I Approach Problems as a Computer Scientist

Throughout this course, I learned that solving problems begins with understanding the objective before writing any code. I break larger problems into smaller pieces, research possible solutions, and test my work frequently. During this project, I first focused on understanding how the maze environment worked before implementing the reinforcement learning algorithm one section at a time. When the model did not initially perform well, I adjusted training parameters, reviewed the algorithm, and continued testing until the pirate consistently reached the treasure. This process reinforced the importance of patience, debugging, and iterative improvement when developing software.

## Ethical Responsibilities

As a computer scientist, I have a responsibility to develop systems that are accurate, reliable, and fair. AI systems can influence important decisions, so developers must consider issues such as privacy, bias, transparency, and security throughout the development process. Even though this project involved a game, the same principles apply to real-world AI applications. Users should be able to trust that AI systems are designed responsibly and that organizations use data ethically. Building software that is dependable and respects the people who use it is an important part of being a professional software developer.
